# 🦷 Cuento de Mati - Cepillarse los Dientes

Un cuento interactivo para niños pequeños que enseña la importancia de cepillarse los dientes de forma divertida.

## 📖 Historia

Mati y su papá terminan de almorzar y descubren que unos simpáticos "bichitos" han llegado a los dientes de Mati. Juntos, aprenderán a cepillarse los dientes y la lengua para mantener una sonrisa limpia y saludable.

## 🎮 Características

- **5 páginas interactivas** con ilustraciones SVG originales
- **Juego de tocar** para eliminar los bichitos de los dientes
- **Efectos de sonido** generados con Web Audio API
- **Animaciones** CSS suaves y divertidas
- **Diseño responsivo** optimizado para tablets y celulares
- **Sin dependencias externas** - funciona offline

## 👶 Edad recomendada

2-4 años (con ayuda de un adulto)

## 🚀 Cómo usar

Simplemente abre `index.html` en cualquier navegador moderno. No necesita servidor ni conexión a internet.
